## Current Features
* Sign up for an account
* Add merch to a cart
* Time you added item to cart
* Delete from cart

## Planned Features
* A checkout page

## Installation

1. Clone repo
2. run `npm install`

## Usage

1. run `node server.js`
2. Navigate to `localhost:5000`
